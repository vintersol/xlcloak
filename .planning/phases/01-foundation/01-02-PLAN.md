  <action>
Create `src/xlcloak/excel_io.py` with three components:

1. **WorkbookReader** class:
   - `__init__(self, path: Path)`: Store path, do not open yet
   - `open(self) -> openpyxl.Workbook`: Call `load_workbook(str(self.path), data_only=False)` and return workbook. Using `data_only=False` preserves formula strings.
   - `iter_text_cells(self, wb: Workbook) -> Iterator[CellRef]`: Iterate all sheets, all rows, all cells. Yield `CellRef(sheet_name=ws.title, row=cell.row, col=cell.column, value=str(cell.value))` for cells where `cell.data_type == 's'` (string type) and `cell.value is not None`.
   - `scan_surfaces(self, wb: Workbook) -> list[SurfaceWarning]`: Scan for unsupported surfaces and return warnings list. Detection logic:
     - **Formulas**: For each cell, check `cell.data_type == 'f'` — create SurfaceWarning with surface_type="formula", detail=str(cell.value), level="WARNING"
     - **Comments**: For each cell, check `cell.comment is not None` — create SurfaceWarning with surface_type="comment", detail=cell.comment.text, level="WARNING"
     - **Charts**: For each worksheet, check `len(ws._charts) > 0` — create ONE SurfaceWarning per sheet with surface_type="chart", detail=f"{len(ws._charts)} chart(s)", level="WARNING", cell=CellRef(sheet_name=ws.title, row=0, col=0)
     - **Merged cells**: For each worksheet, iterate `ws.merged_cells.ranges` — create SurfaceWarning with surface_type="merged_cells", detail=str(range), level="INFO", cell=CellRef(sheet_name=ws.title, row=0, col=0)
     - **Images**: For each worksheet, check `len(ws._images) > 0` — create SurfaceWarning with surface_type="image", detail=f"{len(ws._images)} image(s)", level="INFO", cell=CellRef(sheet_name=ws.title, row=0, col=0)
     - **Data validation**: For each worksheet, check `ws.data_validations` and `len(ws.data_validations.dataValidation) > 0` — create SurfaceWarning with surface_type="data_validation", detail=f"{len(ws.data_validations.dataValidation)} rule(s)", level="INFO"
     - **Named ranges**: Check `len(wb.defined_names.definedName) > 0` — create ONE SurfaceWarning with surface_type="named_range", detail=f"{len(wb.defined_names.definedName)} named range(s)", level="INFO"

2. **WorkbookWriter** class:
   - `__init__(self, source_path: Path, output_path: Path)`: Store both paths
   - `prepare(self) -> None`: Copy source to output using `shutil.copy2(self.source_path, self.output_path)` — this is the copy-then-patch strategy
   - `patch_cells(self, patches: list[tuple[str, int, int, str]]) -> None`: Open the copied workbook, apply patches. Each patch is (sheet_name, row, col, new_value). Set `ws.cell(row=row, column=col).value = new_value` for each. Save workbook.
   - `patch_and_save(self, patches: list[tuple[str, int, int, str]]) -> Path`: Call `prepare()` then `patch_cells(patches)`, return output_path. Convenience method combining both steps.

3. **tests/test_excel_io.py** — write tests FIRST:

   **Round-trip tests:**
   - `test_roundtrip_text_values`: Create workbook with text cells, write, read back, assert values match
   - `test_roundtrip_preserves_numbers`: Create workbook with int/float cells, patch text cells only, verify numbers unchanged
   - `test_roundtrip_preserves_formatting`: Create workbook with bold/colored cells, patch values, verify formatting preserved (check cell.font.bold)
   - `test_patch_replaces_value`: Create workbook, patch one cell, verify new value and other cells unchanged
   - `test_iter_text_cells_skips_numbers`: Create workbook with mixed types, verify iter_text_cells only yields string cells
   - `test_iter_text_cells_skips_none`: Verify empty cells are not yielded

   **Surface detection tests:**

   - `test_detect_comment`: Create workbook with Comment("test note", "author"), scan_surfaces returns WARNING with surface_type="comment"
   - `test_detect_merged_cells`: Create workbook with merged range, scan_surfaces returns INFO with surface_type="merged_cells"
   - `test_detect_no_surfaces`: Clean workbook with only text cells returns empty warnings list
   - `test_multiple_surfaces`: Workbook with formula + comment returns 2+ warnings

   Note: Chart and image detection are difficult to test with openpyxl programmatic creation. Add a comment noting these are validated via the hard fixture in Plan 03.

Date: {iso timestamp}
Mode: sanitize

Sheets processed: {N}

Entity breakdown:
  {TYPE}: {count}

Warnings:
