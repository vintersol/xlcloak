1. Add imports at top: `from rich.console import Console`, `from rich.table import Table`, `from openpyxl.utils import get_column_letter`

2. Add `inspect` command to the existing `main` group:
   ```python
   @main.command()
   @click.argument("file", type=click.Path(exists=True, path_type=Path))
   @click.option("--verbose", is_flag=True, default=False, help="Show confidence scores and detection method")
   def inspect(file: Path, verbose: bool) -> None:
       """Preview what sanitize would do — no files written."""
   ```

3. Inside `inspect`:
   - Create `PiiDetector()`, `TokenRegistry()`, `WorkbookReader(file)`
   - Open workbook, iterate text cells, detect each cell
   - Collect all `ScanResult` objects and `SurfaceWarning` objects
   - **Summary header** per D-13: Print entity counts by type using `click.echo`:
     ```
     xlcloak inspect: data.xlsx
     
     Detected 12 entities:
       EMAIL: 3
       PERSON: 5
       PHONE: 2
       URL: 2
     ```
   - **Per-cell table** per D-13 using `rich.table.Table`:
     - Columns: "Sheet", "Cell", "Type", "Original", "Would-be Token"
     - If `verbose` per D-15: add "Score", "Method" columns (Score from Presidio result, Method = "NER" for PERSON/ORG, "pattern" for EMAIL/PHONE/URL)
     - Truncate Original to 40 chars: `original[:40] + "..." if len(original) > 40 else original`
     - Cell reference: `get_column_letter(cell.col) + str(cell.row)`
   - **Warnings section** per D-14: If warnings exist, print a separate section:
     ```
     Warnings (unsupported surfaces — not sanitized):
       - Sheet1!B5: formula =SUM(A1:A4)
       - Sheet1: 2 chart(s)
     ```
     Use the same format as Manifest._format_warning_line (sheet-level uses row=0/col=0 sentinel)
   - Print "No files written." at the end

4. **NOTE on verbose scores:** The current `detect_cell` returns `list[ScanResult]` which does NOT include confidence scores. For verbose mode, we need scores from Presidio's `RecognizerResult`. Two options:
   - Option A: Extend `ScanResult` dataclass with optional `score: float | None` and `method: str | None` fields
   - Option B: Store scores only for the inspect display path
   
   Choose Option A: Add `score: float | None = None` and `detection_method: str | None = None` fields to the `ScanResult` dataclass in `models.py`. Update `PiiDetector.detect_cell()` in `detector.py` to populate these from the Presidio result: `score=pr.score`, `detection_method="NER" if pr.entity_type in ("PERSON", "ORGANIZATION") else "pattern"`. This keeps data flow clean and avoids duplicating logic.
