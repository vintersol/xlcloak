  <action>
Create `src/xlcloak/bundle.py`:

1. **Constants:**
   - `DEFAULT_PASSWORD = "xlcloak"`
   - `PBKDF2_ITERATIONS = 600_000` (OWASP recommendation per research, resolving STATE.md blocker)
   - `SALT_LENGTH = 16`

2. **`_derive_key(password: str, salt: bytes) -> bytes`** private function:
   - Create `PBKDF2HMAC(algorithm=hashes.SHA256(), length=32, salt=salt, iterations=PBKDF2_ITERATIONS)`
   - Return `base64.urlsafe_b64encode(kdf.derive(password.encode()))`

3. **`BundleWriter` class:**
   - `__init__(self, password: str = DEFAULT_PASSWORD)` — store password, compute `password_mode = "default" if password == DEFAULT_PASSWORD else "custom"`
   - `write(self, path: Path, forward_map: dict[str, str], reverse_map: dict[str, str], original_filename: str, sheets_processed: list[str], token_count: int) -> None`:
     - Build payload dict per D-08:
       ```python
       {
           "version": xlcloak.__version__,
           "original_filename": original_filename,
           "created_at": datetime.now(tz=timezone.utc).isoformat(),
           "sheets_processed": sheets_processed,
           "token_count": token_count,
           "password_mode": self._password_mode,
           "forward_map": forward_map,
           "reverse_map": reverse_map,
       }
       ```
     - Generate 16-byte random salt: `salt = os.urandom(SALT_LENGTH)`
     - Derive key: `key = _derive_key(self._password, salt)`
     - Encrypt: `ciphertext = Fernet(key).encrypt(json.dumps(payload).encode())`
     - Write to file: `path.write_bytes(salt + ciphertext)` — salt prepended per bundle format

4. **`BundleReader` class:**
   - `__init__(self, password: str = DEFAULT_PASSWORD)` — store password
   - `read(self, path: Path) -> dict` — read file bytes, split `data[:SALT_LENGTH]` as salt and `data[SALT_LENGTH:]` as ciphertext, derive key, decrypt, return parsed JSON dict
   - Raise `ValueError("Invalid password or corrupted bundle")` if Fernet decrypt fails (catch `InvalidToken`)

5. **Update `src/xlcloak/__init__.py`:** Add `BundleWriter`, `BundleReader` to imports and `__all__`.

Imports needed:
- `import base64, json, os` from stdlib
- `from datetime import datetime, timezone`
- `from pathlib import Path`
- `from cryptography.fernet import Fernet, InvalidToken`
- `from cryptography.hazmat.primitives import hashes`
- `from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC`
- `import xlcloak` (for `__version__`)
