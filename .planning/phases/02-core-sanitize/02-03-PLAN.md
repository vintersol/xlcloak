**Create `src/xlcloak/sanitizer.py`:**

1. **`derive_output_paths(input_path: Path, output_override: Path | None = None) -> tuple[Path, Path, Path]`** function per D-10, D-12:
   - If `output_override` is provided: `base = output_override.parent / output_override.stem`
   - Else: `base = input_path.parent / input_path.stem`
   - Return `(base.with_name(base.name + "_sanitized").with_suffix(".xlsx"), base.with_suffix(".xlcloak"), base.with_name(base.name + "_manifest").with_suffix(".txt"))`

2. **`check_overwrite(paths: list[Path], force: bool) -> None`** function per D-11:
   - Check each path with `.exists()`
   - If any exist and `force=False`: raise `click.UsageError(f"Output file(s) already exist: {', '.join(str(p) for p in existing)}. Use --force to overwrite.")`

3. **`Sanitizer` class:**
   - `__init__(self, detector: PiiDetector, password: str = DEFAULT_PASSWORD)` — store detector and password
   - `run(self, input_path: Path, output_path: Path | None = None, force: bool = False) -> SanitizeResult`:
     - Derive output paths via `derive_output_paths(input_path, output_path)`
     - Check overwrite protection via `check_overwrite([sanitized_path, bundle_path, manifest_path], force)`
     - Create `TokenRegistry()`
     - Create `WorkbookReader(input_path)`, call `reader.open()`
     - Collect `text_cells = list(reader.iter_text_cells(wb))`
     - Collect `warnings = reader.scan_surfaces(wb)`
     - For each text cell: call `detector.detect_cell(cell, registry)` -> accumulate `all_scan_results` and `patches` list of `(sheet_name, row, col, replaced_text)`
     - Only add to patches if `scan_results` is non-empty (cell had PII)
     - Create `WorkbookWriter(input_path, sanitized_path)`, call `writer.patch_and_save(patches)`
     - Create `BundleWriter(self._password)`, call `writer.write(bundle_path, registry.forward_map, registry.reverse_map, input_path.name, sheet_names, len(registry))`
       - `sheet_names = [ws.title for ws in wb.worksheets]`
     - Create `Manifest(file_name=input_path.name, sheets_processed=len(wb.worksheets), cells_scanned=len(text_cells), cells_sanitized=..., tokens_generated=len(registry))`
       - `cells_sanitized` = number of cells that had at least one detection (not total detections)
     - Call `manifest.add_scan_results(all_scan_results)` and `manifest.add_warnings(warnings)`
     - Write manifest: `manifest_path.write_text(manifest.render())`
     - Return a `SanitizeResult` dataclass with `sanitized_path`, `bundle_path`, `manifest_path`, `token_count`, `cell_count`, `entity_counts`

4. **`SanitizeResult` dataclass** in same file:
   ```python
   @dataclass
   class SanitizeResult:
       sanitized_path: Path
       bundle_path: Path
       manifest_path: Path
       token_count: int
       cells_sanitized: int
       entity_counts: dict[str, int]
   ```

**Create `src/xlcloak/cli.py`:**

1. **Click group:**
   ```python
   @click.group(context_settings={"auto_envvar_prefix": "XLCLOAK"})
   @click.version_option(version=xlcloak.__version__, prog_name="xlcloak")
   def main() -> None:
       """xlcloak -- reversible Excel text sanitization for AI workflows."""
   ```

2. **`sanitize` command:**
   ```python
   @main.command()
   @click.argument("file", type=click.Path(exists=True, path_type=Path))
   @click.option("--password", default="xlcloak", show_default=True, help="Encryption password for the restore bundle")
   @click.option("--output", "output_path", type=click.Path(path_type=Path), default=None, help="Output path for sanitized file (bundle and manifest derive from it)")
   @click.option("--force", is_flag=True, default=False, help="Overwrite existing output files")
   @click.option("--verbose", is_flag=True, default=False, help="Show detailed output")
   def sanitize(file: Path, password: str, output_path: Path | None, force: bool, verbose: bool) -> None:
   ```
   - Import and create `PiiDetector()` and `Sanitizer(detector, password)`
   - Call `result = sanitizer.run(file, output_path, force)`
   - Print summary to stdout:
     - `click.echo(f"Sanitized: {result.sanitized_path}")` 
     - `click.echo(f"Bundle: {result.bundle_path}")`
     - `click.echo(f"Manifest: {result.manifest_path}")`
     - `click.echo(f"Tokens: {result.token_count} unique entities replaced")`
   - If `password == "xlcloak"`: `click.echo("Warning: Using default password. Use --password for real encryption.", err=True)` per D-06
   - Catch `click.UsageError` for overwrite protection (re-raise) and other exceptions (print error, exit 1)
