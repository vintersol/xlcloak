Create `src/xlcloak/detector.py` with the following:

1. **Constants at module level:**
   - `PRESIDIO_TO_ENTITY_TYPE: dict[str, EntityType]` mapping Presidio entity names to xlcloak EntityType:
     - `"EMAIL_ADDRESS"` -> `EntityType.EMAIL`
     - `"PHONE_NUMBER"` -> `EntityType.PHONE`
     - `"PERSON"` -> `EntityType.PERSON`
     - `"URL"` -> `EntityType.URL`
     - `"ORGANIZATION"` -> `EntityType.ORG`
   - `PHASE2_ENTITIES: list[str] = list(PRESIDIO_TO_ENTITY_TYPE.keys())` — always pass this to `analyze()` to avoid getting unwanted entity types (credit cards, IBANs, etc.)

2. **`PiiDetector` class:**
   - `__init__(self, score_threshold: float = 0.4)` — per D-01 aggressive threshold. Store threshold, set `self._analyzer: AnalyzerEngine | None = None` for lazy init.
   - `_get_analyzer(self) -> AnalyzerEngine` — lazy initialization:
     - Check `spacy.util.is_package("en_core_web_lg")`. If not installed, raise `RuntimeError("spaCy model 'en_core_web_lg' not installed. Run: python -m spacy download en_core_web_lg")`.
     - Create `NlpEngineProvider` with config: `{"nlp_engine_name": "spacy", "models": [{"lang_code": "en", "model_name": "en_core_web_lg"}]}`
     - Create `AnalyzerEngine(nlp_engine=provider.create_engine(), supported_languages=["en"], default_score_threshold=self._threshold)`
     - Cache in `self._analyzer` and return.
   - `detect_cell(self, cell: CellRef, registry: TokenRegistry) -> tuple[list[ScanResult], str]` — detect PII in a single cell:
     - Assert `cell.value is not None`
     - Call `analyzer.analyze(text=cell.value, language="en", entities=PHASE2_ENTITIES, score_threshold=self._threshold)`
     - Sort results by `r.start` descending (right-to-left for offset-safe replacement per D-03)
     - For each result: extract `original = cell.value[r.start:r.end]`, map `entity_type = PRESIDIO_TO_ENTITY_TYPE[r.entity_type]`, get token via `registry.get_or_create(original, entity_type)`, create `ScanResult(cell=cell, entity_type=entity_type, original=original, token=token)`
     - Build replaced text by applying replacements right-to-left on a copy of `cell.value`
     - Return `(scan_results_list, replaced_text)` — scan_results in detection order (reversed back to left-to-right for reporting), replaced_text is the fully-substituted cell value

3. **Update `src/xlcloak/__init__.py`:** Add `PiiDetector` to imports and `__all__`.

Key implementation details:
- Import `from presidio_analyzer import AnalyzerEngine, RecognizerResult` and `from presidio_analyzer.nlp_engine import NlpEngineProvider`
- Import `import spacy.util` for the model check
- The method returns BOTH scan results (for manifest/reporting) AND the replaced text (for writing to the sanitized file)
- Scan results returned should be in left-to-right order (reverse the descending sort after replacement) for consistent reporting
